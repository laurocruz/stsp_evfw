This is the code for the Evolutive Framework, proposed by
Pedro Hokama, Mario C. San Felice, Evandro Bracht and Fabio Usberti

To compile this code you need the lemon library 
that may be obtained here:
https://lemon.cs.elte.hu/trac/lemon

and run:

cd Release
make

This will generate the binary to solve the example problem SSTP.
